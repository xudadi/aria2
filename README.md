Aria2 - CLI Metalink/BitTorrent Client
========

使用说明
--------
    下载Aria2.7z
    下载链接：https://github.com/xudadi/aria2/blob/main/win-download-run/Aria2.7zAria2.7z在电脑上运行
    运行 aria2.exe
    访问 http://xudadi.cn/aria2 查看状态
    打开AriaNg 设置》改RPC (xudadi.cn:6800)为
    1、Aria2 RPC 别名 127.0.0.1:6800
    2、Aria2 RPC 地址 127.0.0.1
    下载后的文件保存在 D盘 文件夹中

文件说明
--------
    Aria2Data     # 下载目录 默认下载文件会保存在这里
    aria2.conf    # 配置文件 可以自己根据说明修改
    aria2.exe     # 启动文件 使用这个来启动 aria2
    aria2.session # 任务保存文件 未完成任务会保存在这里
    aria2c.exe    # 命令行主程序
    README.md     # README
